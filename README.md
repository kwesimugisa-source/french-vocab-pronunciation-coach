# French Vocabulary + Pronunciation Coach

This is a complete starter package for the second app.

## Included
- Next.js App Router setup
- TypeScript
- Tailwind CSS
- Working `@/` import alias
- Static scaffold UI with `app/page.tsx`

## Run
```bash
npm install
npm run dev
```
